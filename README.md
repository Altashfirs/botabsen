# Bot Absen — Attendance Bot

An automated attendance tracking bot system.

## ✨ Features

- Automated attendance recording
- Attendance reports
- Bot-based interface

## 🛠 Tech Stack

- **Language:** Python
- **Bot Framework:** appropriate framework

## 🚀 Getting Started

```bash
git clone https://github.com/Altashfirs/botabsen.git
cd botabsen
pip install -r requirements.txt
python bot.py
```
